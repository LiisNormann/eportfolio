Website for Tartu Guides Association

Project information

  • Category: Web / Wordpress
  • Client: Tartu Guides Association
  • Project date: May, 2022
  • Project Tasks: development, project management, testing, documentation, client relations, design

With a team of two developers, the project comprised of creating a new website for Tartu Guides Association. The task was to make a website which, in comparison to the clients old website, is in accordance with modern web design trends. In addition, the new site had to be made using a different CMS to their old choice.

The two developers shared equally split responsibility for all the project tasks listed above.
